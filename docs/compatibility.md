Compatibility
=============

ical2jcal is compatible with [RFC 5545] and [RFC 7265].
More RFCs are supported by the [icalendar] package.
Please view the [icalendar] documentation for more information.

[icalendar]: https://icalendar.readthedocs.io/en/latest/reference/design.html
[RFC 5545]: https://tools.ietf.org/html/rfc5545
[RFC 7265]: https://tools.ietf.org/html/rfc7265
